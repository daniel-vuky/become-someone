Tools:

**1. ping**
Live host check
```
ping <ip><dns>
```
**2. tracert**
Check the direction of the request over the network
```
tracert <ip>
```

**3. dnsrecon (kali)**
Exploit the DNS record of the domain
```
dns -a -d <dns>
```

**4. Nmap**
Exploit port, os, version, vulnerability
```
nmap <dns>
```

**5. Naabu**
Scan port
```
naabu -p 1-100 -host <dns>
```
